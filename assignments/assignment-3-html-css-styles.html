Box-sizing: border-box
Padding is included in box size.

Box-sizing: default;
Padding adds to box size.

Absolute positioning conforms to the specified edge of the parent element. Relative positioning falls at the beginning of the parent element, usually top-left.

Border-box sizing includes the padding and border in the box's height and width parameters, and will not expand the box beyond specified sizing. Default box-sizing allows padding and border to expand the box's width and height.
